Description du Projet

Ce projet était en deux étapes :
- Faire une gestion d'une base de données en fonction d'un cahier des charges
- Réaliser un site web pour répondre aux besoins d'un client

Intro

Le projet consistait à concevoir un système de gestion de conférences et de conférenciers dans différentes salles pour un client. Cela impliquait la création d'une base de données conforme à un modèle conceptuel de données (MCD) et un modèle logique de données (MLD), ainsi que le développement d'un site web pour interagir avec cette base de données.

Analyse des besoins

Nous avons commencé par une analyse approfondie des besoins du client, en identifiant les fonctionnalités clés du système. Cela comprenait la gestion des salles de conférence, la planification des conférences, la réservation de salles, la gestion des participants et des intervenants, ainsi que la génération de dons.

Conception

Nous avons élaboré un modèle conceptuel de données (MCD) pour représenter les entités principales du système et leurs relations. Ensuite, nous avons traduit ce modèle en un modèle logique de données (MLD), en définissant les tables, les attributs, les clés primaires et étrangères, ainsi que les contraintes d'intégrité.

Développement

Parallèlement à la mise en place de la base de données, nous avons réalisé le développement du site web en intégrant la base de données. Nous avons utilisé des technologies web modernes telles que HTML, CSS, JavaScript et PHP (ou d'autres frameworks comme Bootstrap) pour créer une interface conviviale permettant aux utilisateurs de gérer les conférences, les salles et les participants.
